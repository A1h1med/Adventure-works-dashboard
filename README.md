# Adventure Works Sales Analysis

## Project Overview
This project analyzes Adventure Works sales data using Power BI to create an interactive dashboard that supports business decision-making and performance monitoring.

### Project Objectives
The dashboard provides insights into:
- Sales performance across multiple years.
- Key business KPIs, including Revenue, Profit, Orders, and Customers.
- Year-over-year profit comparison and identification of the highest profit achieved.
- Number of products sold by category.
- Revenue and profit trends over time.
- Geographic distribution of sales across territories using an interactive map.
- Employee performance monitoring, including Top 5 Sales Representatives and employee-specific details such as vacation hours through search functionality.

## Tools
- Power BI
- SQL

## Key Insights
- 2013 recorded the highest sales performance across all years.
- Australia generated the highest number of orders among all sales territories.
- Current profit has surpassed the historical minimum profit but has not yet reached the highest profit achieved.
- The number of customers has shown consistent growth over the years.
- The Bikes category generated the highest revenue compared to all other product categories.
- More than 28K orders were shipped to home addresses, significantly exceeding orders delivered to main office locations.
- Michael was the top-performing sales representative and led the Top 5 Sales Employees ranking.
